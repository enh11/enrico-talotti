About me
--------

I'm a young researcher at the [Center for Information Technologies and Applied Mathematics](https://ung.si/en/research/centre-for-information-technologies-and-applied-mathematics/) at the University of Nova Gorica, Slovenia. My research focuses on algebra, particularly hypercompositional algebra, with an emphasis on its implementation in the Rust programming language. I'm also interested in the algebraic applications of cryptography.

Repositories
------------

[**schnorr\_sigma\_protocol**](https://github.com/enh11/schnorr-sigma-protocol)

An asynchronous client-server implementation of the Schnorr identification protocol in Rust, featuring elliptic curve cryptography, secure serialization of curve points and scalars, and a clean protocol design with prover and verifier roles.

[**weak\_keys\_analysis**](https://github.com/cysecud/ecc_weak_keys)

A pure Rust implementation of several NIST elliptic curves, featuring an implicit version of the Baby Step-Giant Step algorithm for solving the Discrete Logarithm Problem. Implementations are also available in PARI-GP.

[**hypercompositional\_structures**](https://github.com/enh11/hypercompositional_structures)

A pure Rust library containing utilities for dealing with hypercompositional structures. Currently under development.
